# Obsidian Tab Groups (Chrome-Style)

[English](./README.md) | 한국어

크롬(Chrome) 브라우저 스타일의 직관적인 탭 그룹화, 컬러 매핑, 그리고 접기/펼치기 기능을 옵시디언 작업 공간에 이식하는 플러그인입니다.

복잡하게 나열된 탭 바를 관련 노트끼리 묶어 고유의 라벨(이름표)과 색상을 지정하고, 클릭 한 번으로 간편하게 접거나 펼쳐서 화면 공간을 최적화해 보세요.

## ✨ 주요 기능 (개발 중)
---
* **시각적 그룹화 (컬러 매핑 및 라벨):** 특정 탭 묶음에 고유한 색상과 커스텀 이름을 부여합니다. 같은 그룹에 속한 탭 상단에 동일한 컬러 라인이 이어져 시각적으로 연결됩니다.
* **동적 인터랙션 (탭 접기/펼치기):** 그룹 라벨을 클릭하면 소속된 모든 탭이 부드럽게 숨겨져 공간이 절약됩니다. 다시 클릭하면 원래 순서 그대로 나타납니다.
* **상태 보존 (State Persistence):** 옵시디언이 재시작되거나 창 배치가 바뀌어도 이전에 구성해 둔 탭 그룹 설정과 접힘/펼침 상태가 그대로 유지됩니다.
* **테마 호환성:** 라이트/다크 모드는 물론 다양한 커뮤니티 테마의 스타일을 해치지 않도록 투명도 기반의 색상 처리가 적용됩니다.

## 🛠️ 설치 방법
---
### 수동 설치 Guide
본 플러그인은 현재 활발히 개발 중인 단계로, 공식 커뮤니티 플러그인 카탈로그 등록 전까지는 다음과 같이 수동으로 설치하여 테스트할 수 있습니다.

1.  배포된 최신 릴리즈 파일(`main.js`, `manifest.json`, `styles.css`)을 다운로드합니다.
2.  옵시디언 보관함(Vault)의 숨김 폴더인 `.obsidian/plugins/` 경로로 이동합니다.
3.  `obsidian-tab-groups` 폴더를 새로 만들고 다운로드한 파일들을 넣습니다.
4.  옵시디언 **설정 > 커뮤니티 플러그인** 메뉴에서 **Tab Groups (Chrome-Style)** 플러그인을 활성화합니다.

## 💻 개발자 가이드
---
본 프로젝트는 옵시디언 공식 샘플 플러그인 구조를 기반으로 빌드되었습니다.

### 시작하기
1. 테스트 보관함의 플러그인 폴더 안에 본 레포지토리를 클론합니다.
2. 필요한 개발 의존성 패키지를 설치합니다:
    ```bash
    npm install
    ```
3. 실시간 코드 변경사항 반영(Hot-reloading)을 위해 개발 서버를 실행합니다:
    ```bash
    npm run dev
    ```
